This is a description of the different pages in this repository.
Please also preview with a browser for a visual demo.
These websites are also responsive, which means they look good on a laoptop or a phone.

index.html: this is the main page, where you click the button to log in.

login.html: this is the location that you sign in with a Username and a Password. (Note: this is a visual prototype,
so there are no actual credentials required.) When you sign in, you will be taken to the search page.

search.html: this page has a list of all of the students.(note: the names are just for testing.) in the future,
we hope to be able to link into our school's database and display actual student's info.

student.example.html: this is the page that will display the students info.
